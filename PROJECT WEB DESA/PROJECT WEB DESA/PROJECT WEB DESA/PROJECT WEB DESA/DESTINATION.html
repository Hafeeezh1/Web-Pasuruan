Alun-Alun Pasuruan

Alun-alun kota Pasuruan identik dengan sebuah taman yang asri, dimana terdapat beberapa tanaman hias, rerumputan, bangku taman dan jogging track. Alun-alun ini ramai dikunjungi saat pagi dan sore hari, dan akan semakin ramai saat diadakannya gelaran acara di sekitar alun-alun. Pelabuhan Pasuruan

Pelabuhan Pasuruan tidak hanya menjadi tempat perahu nelayan bersandar. Kawasan ini juga disulap menjadi lokasi wisata. Pengunjung yang datang ke tempat ini rata rata ingin menikmati debur ombak dan angin laut di dermaga.

Gedung Harmonie

Gedung ini memiliki dua lantai dan memiliki kubah di bagian atasnya. Gedung Harmonie Pasuruan dulunya digunakan sebagai tempat pertemuan dan pertunjukan musik. Gedung ini sekarang menjadi museum dan dibuka untuk umum.
